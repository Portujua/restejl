## Prerequisites ##

 - [PHP (>= 5.6)](http://windows.php.net/download#php-7.1)
 - [Composer](https://getcomposer.org/download/)

## Setup ##

 - Open command line and run 

> composer install

### Adding routes ###
In order to add new routes you must add a file inside:

> src/routes

There is an example called **users.php**

### API URL ###

> *url/api/**yourService***
